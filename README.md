# Project Overview

A simple blogging platform with user authentication and content management.

## Features

-   **User Authentication**: Signup, Login, Logout
-   **Posts CRUD**:
    -   Create, Read, Update, Delete (users can only edit/delete their own posts)
    -   Cover image upload on create/edit (optional on edit)
    -   Pagination
    -   Change view: list/grid
-   **Post Page**:
    -   Update and delete from the post page
-   **Comments**:
    -   Add, Read, Delete (users can delete their own comments or comments on their posts)
    -   No edit for comments
-   **User Panel**:
    -   Change username
    -   Change password
    -   List of user posts with action buttons
